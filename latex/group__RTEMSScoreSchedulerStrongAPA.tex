\subsection{Detailed Description}

This is an implementation of the Strong A\+PA scheduler defined by Bradenbug et. al in Linux\textquotesingle{}s Processor Affinity A\+PI, Refined\+: Shifting Real-\/\+Time Tasks Towards Higher Schedulability.
